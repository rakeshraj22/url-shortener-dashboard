# Full-Stack URL Shortener & Analytics Dashboard

A scalable full-stack URL shortening platform with analytics dashboard, Redis caching, and Dockerized deployment.

## 🚀 Tech Stack

- Angular
- TypeScript
- Java
- Spring Boot
- PostgreSQL
- Redis
- Docker

## ✨ Features

- Generate short URLs using Base62 encoding
- Redirect short URLs to original links
- Redis caching for fast redirects
- Analytics dashboard
- Total clicks / top links
- Docker Compose multi-service setup

## 🐳 Run with Docker

```bash
docker compose up --build
